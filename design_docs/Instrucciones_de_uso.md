# ¿Cómo utilizar estos documentos?
**Guía para el equipo de diseño (No técnico)**

Este paquete de documentos contiene los textos reales ("copy") que el usuario verá en el sitio web de INVEST El Salvador.

## Objetivo
El propósito es que al diseñar las pantallas (mockups), utilices el texto **real** en lugar de "Lorem Ipsum", para asegurar que el diseño se adapte al contenido verdadero (longitud de textos, idiomas, botones).

## Estructura de los archivos
Encontrarás un archivo para cada página o componente principal, por ejemplo:
*   `TA_home.md` (Página de Inicio)
*   `TA_header.md` (Menú de navegación superior)
*   `TA_footer.md` (Pie de página)

## Instrucciones paso a paso

1.  **Abre el archivo** correspondiente a la pantalla que estás diseñando.
2.  **Columna Multimedia:**
    *   Esta es tu herramienta de comunicación.
    *   En lugar de `[Nombre_de_archivo_de_multimedia]`, escribe el nombre del archivo de imagen, icono o video que propones usar ahí.
    *   *Ejemplo:* Si en el Hero va una foto de surf, escribe `foto-surf-hero-v1.jpg`.
3.  **Columnas de Texto:**
    *   Copia y pega el texto de la columna **Texto (ES)** o **Texto (EN)** directamente en tu programa de diseño (Figma, Photoshop, etc.).
    *   Esto garantiza que no haya errores de tipeo y que se respete el contenido aprobado.
4.  **Columna Notas:**
    *   Lee esta columna para entender contextos (ej. "Este es un botón", "Título principal H1").

## Ejemplo
Si ves esta fila:

| Multimedia (Archivo) | Sección | Texto (ES) |
| :--- | :--- | :--- |
| `icono-avion.svg` | Propuesta 1 | Ubicación Estratégica |

Significa que debes colocar el icono `icono-avion.svg` junto al título "Ubicación Estratégica".
